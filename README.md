# Customer Behavior Dashboard (Power BI)

This project presents an interactive *Power BI dashboard* that analyzes customer behavior, purchasing patterns, and sales performance using an e-commerce dataset.  
The dashboard provides key insights into customer demographics, subscription status, category-wise revenue, and age-based purchasing trends.
